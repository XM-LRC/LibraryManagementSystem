-   配置数据库信息：通过navicat等软件建立名为db_library的数据库，注意使用utf-8编码（如下图），然后导入源码文件夹里的db_library.sql文件到刚才建的数据库。

- ![image-20200630182607618](readermeImages/配置1.png)

- ![image-20200630182643180](readermeImages/配置2.png)
